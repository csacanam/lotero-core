# Contributing to Lotero

Thanks for your interest in contributing.

## How to contribute

- Open an [issue](https://github.com/csacanam/lotero-core/issues) for bugs or feature requests
- Fork the repo, create a branch, and open a [pull request](https://github.com/csacanam/lotero-core/pulls)

## Development

```bash
yarn install
yarn chain    # Terminal 1
yarn deploy   # Terminal 2
yarn hardhat:test
yarn start    # Optional: frontend
```

See [README.md](README.md) for details.
